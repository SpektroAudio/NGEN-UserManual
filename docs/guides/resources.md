# Addditional NGEN Resources

## NGEN Web Manager

The NGEN Web Manager is an open-source web application that lets users manage files stored in NGEN's internal memory and microSD card, debug the device, and load example projects into NGEN's memory.

[https://github.com/SpektroAudio/NGEN-Web-Manager](https://github.com/SpektroAudio/NGEN-Web-Manager)
